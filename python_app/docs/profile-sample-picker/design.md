## UX

### Profile → Image (Profile overrides)
- Background samples dir:
  - Browse / Open / Reload
  - Below it: selectable list of images from that folder (max 5)
  - “Random” checkbox disables the list
- Thumbnail samples dir:
  - Browse / Open / Reload
  - Below it: selectable list of images from that folder (max 5)
  - “Random” checkbox disables the list

### Manual vs Random
- Random checked: list is disabled and selection cleared
- Random unchecked: user selects up to 5 from the list (manual)
